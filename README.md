## About The Project

This project just for learning React Native :)


### Built With
* React
* React Native


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Rizki36/AudioBook.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run project
   ```sh
   npm run ios
   // or
   npm run android
   ```
